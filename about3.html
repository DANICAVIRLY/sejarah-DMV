Direction

HTML

HTML (Hypertext Markup Language) adalah bahasa yang digunakan untuk membuat struktur halaman web. HTML menggunakan tag untuk menentukan elemen-elemen seperti teks, gambar, tautan, dan lainnya.

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web. CSS memungkinkan Anda untuk mengubah warna, font, layout, dan elemen-elemen visual lainnya pada halaman HTML.

JAVA SCRIPT

JavaScript (JS) adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. JS memungkinkan elemen-elemen di halaman web untuk merespons tindakan pengguna, seperti klik, pengisian form, atau pergerakan mouse.
